# Quality Assurance (QA)

This folder contains ESAPI scripts for quality assurance workflows and measurements.

## Purpose

QA scripts automate routine quality assurance tasks and help maintain treatment delivery accuracy.

## Typical QA Tools

- Daily QA automation
- Monthly QA checks
- Annual QA procedures
- Machine QA data collection
- Patient-specific QA plan generation
- QA result analysis

## Safety Notice

⚠️ **IMPORTANT**: QA scripts assist in quality assurance workflows but do NOT replace regulatory requirements or standard operating procedures. All QA must be performed according to institutional protocols and regulatory guidelines.
