# Plan Checks

This folder contains ESAPI scripts for automated treatment plan validation and quality checks.

## Purpose

Plan check scripts help ensure treatment plans meet clinical protocols and safety standards before delivery.

## Typical Checks

- Dose constraints validation
- Plan normalization verification
- Field parameter checks
- MU calculations review
- Collision detection
- Treatment technique verification

## Safety Notice

⚠️ **IMPORTANT**: These scripts are provided as tools to assist in clinical workflow but do NOT replace professional clinical judgment and manual verification. All treatment plans must be reviewed and approved by qualified medical physics and radiation oncology staff.
